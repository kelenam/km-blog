### none
`none` returns `true` if the provided predicate function returns `false` for all elements in a collection, `false` otherwise.

Use `Array.prototype.some()` to test if any elements in the collection return `true` based on `fn`. Omit the second argument, `fn`, to use `Boolean` as a default.

```js
const none = (arr, fn = Boolean) => !arr.some(fn);

Examples:
none([0,1,3,0], x => x == 2) // true
none([0,0,0]); //--> true
none([0,0,1]); // --> false
```

### When / Why would you use this?
Well, it's the exact same function as `any` except negated so likely in conditions where the inverse of that use-case existed.
