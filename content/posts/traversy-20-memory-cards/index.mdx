## Key Insights
So I was actually having an issue running `npx create-react-app` and having an error related to node_modules and npm, the reason for this was that I had node_modules in my user root directory. 

On our card class we are adding the transforms to control the card rotation and the slide effect. 

Handling rotation is down with a class `show-answer`, that will trigger our card to trigger its animation on the x axis and "flip".

backface-visibility: hidden is a property we use to hide what is on the backface side of something.

The transform-style CSS property sets whether children of an element are positioned in the 3D space or are flattened in the plane of the element
https://developer.mozilla.org/en-US/docs/Web/CSS/transform-style

Font awesome has "unit-codes" that can be used within our css to insert an icon. We will be using f021. The way that you use it is by saying `content: '\f021 Flip'` Then setting the `font-family: 'Font Awesome 5 Free', 'Lato'`,
