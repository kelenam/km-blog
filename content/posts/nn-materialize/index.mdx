## Setting Up
- Need to add Materialize to project,
- We'll use the CDN, and include in the head.

```html 
<!-- Compiled and minified CSS -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/css/materialize.min.css">

<!-- Compiled and minified JavaScript -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/js/materialize.min.js">
```            

Also going to import the google font icon library:
```html 
  <!--Import Google Icon Font-->
<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
```
- This will give use icons in our project

## Basic Text Styles 
- Materialize comes with some default styles 
- Let's create a wrapper container, with a class of `container`, a container is special class in Materialize with its own styles...

- Most elements are using REMs as units.
>[^ **What are `rem`s? How are they measured?**]- **REMs** Root Element, if our document (html) has a font-size of 14px, what we are going to do is take that root element value of the font size then we are going to be a muliple of that root value for our text elements, so an h1 with a `font-size: 4.2rem;` will be 4.2x the size of the base root style.

### p `flow-text`
Giving a `p` tag a class of `flow-text`, gives them auto-resizing ability based on screen size, as the screen gets bigger/smaller so does the text. It's useful for things like introductions to paragraphs or maybe first lines of a blog?

## Hiding Content
```html 
<div class="container">
    <h4 class="hide">Hide me on all screen sizes</h4>
    <h4 class="hide-on-small-only">Hide me on small screen sizes</h4>
    <h4 class="hide-on-med-only">Hide me on medium screen sizes</h4>
    <h4 class="hide-on-large-only">Hide me on large screen sizes</h4>
    <h4 class="hide-on-med-and-down">Hide me on medium & small screen sizes</h4>
    <h4 class="hide-on-med-and-up">Hide me on medium & large screen sizes</h4> 
</div>
```
- Assigning a class of ...
    - `hide` will hide the whole element
    - `hide-on-small-only` hide only small screens, smll tabs and mobile
    - `hide-on-med-only` hide only med screens, tablets
    - `hide-on-large-only` hide on large screens, tablets
    - `hide-on-med-and-down` hide on medium sizes and below
    - `hide-on-med-and-up` hide on medium sizes and up
    
## Colors
We have a huge range of colors for materialize 

- To change background colors we can give a class of the color name, to make it darker or lighter we can say darken or lighten. 
- `blue lighten-4`, for the color lighter color range.
- We could also do something like `purple darken-4`
- To change text color : `red-text` 

We can also darken or lighten the text:
- `purple-text text-darken-2`
- The number values relate to the degree to darken/lighten, like 2x, 3x, etc.

> Thing to remember, just specify a color for changing the bg color, say {colorName}-text for text color.
> use lighten-{intensity} or text-lighten-{intensity} to lighten and darken for darken

## Responsive Grid
```html 
<div class="container">
    <div class="row"> 
        <div class="col s3">content</div> 
        <div class="col s3">content</div> 
        <div class="col s3">content</div> 
        <div class="col s3">content</div> 
    </div>
    <div class="row"> 
        <div class="col s12 m6">content</div> 
        <div class="col s12 m6">content</div> 
        <div class="col s12 m6">content</div> 
        <div class="col s12 m6">content</div> 
    </div>
    <div class="row"> 
        <div class="col s12 m6 l4">content</div>  
        <div class="col s12 m6 l4">content</div>  
        <div class="col s12 m6 l4">content</div>  
        <div class="col s12 m6 l4">content</div>  
    </div>
    <div class="row"> 
        <div class="col s12 m6 l4 xl2">content</div>  
        <div class="col s12 m6 l4 xl2">content</div>  
        <div class="col s12 m6 l4 xl2">content</div>  
        <div class="col s12 m6 l4 xl2">content</div>  
    </div>
    <div class="row"> 
        <div class="col s12 m8 ">content</div>  
        <div class="col s12 m4 ">content</div>  
        <div class="col s12 m4 ">content</div>  
        <div class="col s12 m8 ">content</div>  
    </div>
</div>
```
- we need a class of `row` surrounding our elements
- 12 columns grid, `col s1` would be a col taking up 1 unit on small screen sizes and up.
- every item within a row must have a `col` class.
- The default behavior is sizing elements based on content, we can instead specify the size we want, `col s3` would mean an item taking up 3 columns in width (3 of 12) on small screen sizes and up.

- What if we want to give different widths, say for small size and medium sizes? `col s6 m3`, will be 6 cols for small and up until it hits medium size screens which it will then be m3.
- Better to work from the small screens upwards, starting with s sizes, 
- we can do that for all sizes: `col s12 m6 l4`
- for xl screens, we can make it two columns in width: `col s12 m6 l4 xl2`

They don't have  to have the same number either.

## Depth & Shadows
We can control drop shadow using only one class, it's: `z-depth-1`, 1 is the smallest number. 4 and 5 get a little unreleastic looking, 1-3 are more functional.
